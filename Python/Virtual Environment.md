#venv #Python

### 1. Navigate to Project Directory
``` bash
cd path_to_project_dir
```

### 2. Create Virtual Environment
``` bash
python -m venv venv
```

### 3. Activate the Virtual Environment
``` bash
# Windows
venv\Scripts\activate

# macOS\Linux
source venv/bin/activate
```

### 4. Install Packages
``` bash
pip install ...
```

### Deactivate Virtual Environment
``` bash
deactivate
```
